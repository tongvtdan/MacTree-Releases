# MacTree Beta Releases

This repository hosts signed MacTree beta downloads, the Sparkle update feed, release notes, and tester feedback. The application source remains private.

MacTree is a native macOS file workspace for navigating folders, inspecting files, and performing common local file actions without losing context.
