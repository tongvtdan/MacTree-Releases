# Install MacTree Beta

MacTree requires macOS 14 or later and supports both Apple silicon and Intel Macs.

1. Download the latest `MacTree-*.zip` from Releases.
2. Open the ZIP and move `MacTree.app` to Applications.
3. Open MacTree and choose a folder when prompted.
4. Use **MacTree > Check for Updates…** to verify the update feed.

Only install assets published by this repository. Production beta assets are signed with a Developer ID Application certificate and notarized by Apple.
