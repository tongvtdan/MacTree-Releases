# MacTree Beta Privacy

MacTree works locally with folders that you choose or explicitly allow through macOS.

- MacTree does not require an account.
- MacTree does not upload filenames, paths, file contents, usage events, or diagnostic data.
- File-access bookmarks remain on your Mac so previously chosen folders can be reopened.
- Diagnostic reports are generated locally, omit paths and filenames, and are shared only when you copy and send one yourself.
- Update checks contact this GitHub-hosted Sparkle feed and GitHub release assets.

Beta feedback submitted through GitHub is governed by GitHub’s privacy terms and is visible according to the issue visibility of this public repository. Do not include private filenames, paths, file contents, or personal information in feedback.
